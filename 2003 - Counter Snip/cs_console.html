Les commandes les plus importantes :

Voici toutes les commandes à taper dans la console intéressante.

Attention sous counter-strike le clavier est en qwerty et donc :
Au lieu de taper a z m . / - ! _
Il faut taper q w , : ! ) shift +"1" shift + ")"

cl_dynamiccrosshair "0"
En rajoutant cette commande dans votre config.cfg, vous obtiendrez une croix qui ne s'écartera plus dès que vous marcherez :)

adjust_crosshair
Change la couleur de la réticule de visée ( il y a 5 couleurs qui tournent )

con_color xxx.xxx.xxx
Change la couleur du hub

cl_showfps "1"
Un petit texte vous affiche seulement les FPS en haut à gauche

name X
Pour changer de nom durant une partie (il faut être vivant pour taper cette commande). Dans la console il vous faut mettre le signe"%" à la place d'un espace seulement entre les noms

hud_centerid X
Centrer les textes lorsque le statut du joueur en face est écrit, "0" pour les afficher par défaut (c'est-à-dire en bas à gauche), "1" pour les centrer (gênant car lorsque vous viser votre adversaire un texte s'affiche au milieu et vous empêche de bien viser).

xshowradar
"-" pour désactiver le radar, "+" pour l'activer
Ne marche plus !

timeleft
Pour connaître le temps restant avant la fin du round

cl_hidefrags X
Cache les scores des autres joueurs : 1=activé, 0=desactivé

cl_righthand X
arme à droite : 1=activé, 0=desactivé

max_shells X
Nombre maximum de douilles éjectées par l'arme d'un joueur

fastsprites X
Détails de la fumée des grenades SG (fumigènes), de "0" à "2" (0 pour les détails par défaut, "2" pour très peu de détails)

max_smokepuffs X X
Nombre de détails de la fumée d'une grenade SG

listmaps
Montre la liste des maps pouvant être exécutées par le serveur ou choisies par le serveur

votemap X
Vote pour la map que vous voulez faire le round d'après

listplayers
Montre la liste de tous les joueurs participant à ce round et le numéro qui leur est attribué

setinfo ah X
Active l'affichage de l'aide en cours de partie : 1=activé, 0=desactivé

setinfo dm X
Active l'affichage des infos sur les maps : 1=activé, 0=desactivé

status
Affiche la liste de tous les joueurs plus leurs temps de latence,frame rate...

ping
Affiche le ping de tous les joueurs

info X
Liste les infos d'un joueur

cvarlist
Liste les variables d'un serveur

adjust_crosshair XXX
Change la couleur du curseur. Taper un nombre (ex: si on tape adjust_crosshair 001 on obtient un curseur vert)

kill
Pour vous suicidez

net_graph 2
Affiche un graphique pour que vous puissiez vois vos images par secondes... (il y a aussi net_graph 1 ou 3 ou....)

max_fps 100
Augmente le nombre d'images par secondes (très pratique, on vise mieux)

rate 3000
Réduit un peu le ping, vous pouvez aller jusqu'à 6000 mais après ça fait beaucoup d'upload pour ceux qui ont le câble par Noos

exit ou quit
Quitter entièrement Counter-strike

r_drawviawmodel X
1 pour activer l'arme visible devant vous ou 0 pour qu'il n'y ait plus d'arme devant vous afin de gagner en visibilé

sv_gravity
Change la gravité (800 par défault, je conseille 075 ou 125 )

changelevel de_dust
Change la map en cour (remplacer de_dust par la map voulue)

nextmap
Map suivante sur le serveur

mp_friendlyfire x
Tir fraternel ( 0 activer ; 1 désactiver )

cl_bob 0
Evite que l'arme bouge lorsque vous courez

mp_c4timer x
Change le temps que met la bombe avant d'exploser ( 35 par default )

mp_startmoney x
Change la quantité d'argent que vous aurez au départ ( défault : 800 )

mp_freezetime x
Change le temps au début de chaque round pour acheter ses armes ( défault : 6 )

mp_autoteambalance x
Pour équilibrer les équipes automatiquement = 1 ; sinon = 0

mp_fadetoblack X
Si X vaut 1, les joueurs ont un écran noir dès qu'ils sont morts.
Cela évite tout triche. Les joueurs morts peuvent continuer à chatter et voir les scores.

mp_limitteams X
X vaut le nombre de joueurs qu'une équipe peut posséder en plus de sa rivale. Si X vaut zéro, cette option est désactivée (default = 2 )

mp_footsteps X
Si X=1, le son des pas est active. Si X=0, les joueurs marchent sans faire de bruit.

mp_flashlight X
Autorise (X=1) ou pas (X=0) l'utilisation des lampes-torches.
